River Rave

- Enter password "fansoft"

Restoring your Savegames
 Update the last to the current Release
 

Space Invaders, Moon Patrol, Total Tron & River Rave

- Copy the file HISCORE.BIN in your game folder to a save place

- Install the new Version

- Restore the file HISCORE.BIN in your new game folder

Donald Duck

- Copy the file HISCORE.BIN + DONALD.BIN in your game folder to a save place

- Install the new Version

- Restore the file HISCORE.BIN + DONALD.BIN in your new game folder

DeWereld

- Copy the file USER.MDB in your game DATA folder to a save place

- Install the new Version

- Restore the file USER.MDB in your new game DATA folder
